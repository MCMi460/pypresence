.. py:class:: ActivityType

 Enum of activity types `(supported by Discord RPC) <https://discord.com/developers/docs/game-sdk/activities#data-models-activitytype-enum>`_.

  :PLAYING: The default activity type and displays a "playing" status.

  :LISTENING: Displays a "listening" status.

  :WATCHING: Displays a "watching" status.

  :COMPETING: Displays a "competing" status.
